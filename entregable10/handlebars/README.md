## Instalacion

Requiere [Node.js](https://nodejs.org/) v16+ para correr.

Instale las dependencias y ejecute la aplicacion.

```sh
npm install
node index.js
```

## Correr Aplicacion
Abra su navegador y escriba la siguiente direccion.

```sh
http://localhost:8080
```
